# ebabel-distance
Return the distance in 3D space from point "a" x, y, and z coordinates to point "b" x, y, and z coordinates.

## Install
```
npm i --save ebabel-distance
```

## Usage
```
const distance = require('ebabel-distance');

const result = distance([-4, 10, 3], [2, 3, 8])
```

## Develop this module
```
npm install
```

## Test this module
